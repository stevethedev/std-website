# Steve the Dev Website

This repo contains the code for the Steve the Dev website. It is divided into
the following sections:

1. **Server API** (`server-api`) provides a RESTful API to the clients.
   
2. **Public Client** (`client-public`) provides an SPA for the general-public.

3. **Admin Client** (`client-admin`) provides an SPA for administrators.

4. **Shared Client Content** (`client-common`) provides shared content between
   the other `client-*` sections.
